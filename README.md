# Synera - Auto Chess Arena

基于 Qt/C++ 的自走棋对战游戏。玩家购买英雄放置到棋盘上，与敌方自动战斗。

## 操作方式

| 操作 | 说明 |
|------|------|
| 鼠标拖拽 | 从商店/回收槽拖拽英雄到棋盘（玩家半场 y≥4） |
| 拖拽到商店 | 将英雄卖回商店（返还到库存） |
| 拖拽到回收槽 | 将英雄存入回收槽备用 |
| 开始战斗按钮 | 点击右侧按钮开始本关战斗 |
| F5 | 保存游戏进度 |
| F9 | 读取游戏进度 |
| R | 重置游戏 |

## 英雄类型

| 类型 | HP | 攻击 | 射程 | 移速/攻速 | 技能 |
|------|-----|------|------|-----------|------|
| 战士 Warrior | 100 | 20 | 1 | 30/60 | 对最近敌人造成伤害 |
| 法师 Mage | 50 | 10 | 4 | 30/60 | 周围5×5敌方全体燃烧4回合 |
| 辅助 Support | 80 | 0(治疗20) | 1 | 30/60 | 全场HP最低2个单位回复 |
| 刺客 Assassin | 15 | 50 | 1 | 30/40 | 瞬移到最近敌方旁造成伤害 |

## 升星机制

两个**同名且同星级**的英雄可以合成为更高星级的英雄：

- 拖拽一个英雄到棋盘上另一个同名同星英雄的格子上即可升星
- 拖拽一个英雄到回收槽中同名同星英雄上也可升星
- 最高可升至 **3 星**（★→★★→★★★）
- 升星后属性大幅提升：HP、攻击力、技能效果均随星级增长
- 刺客升星后初始法力值满格

## 存档系统

- **F5** 保存当前进度（准备阶段可用），存档文件为 `savegame.json`
- **F9** 读取存档（准备阶段可用）
- 存档内容包括：金币、关卡、HP、商店库存、棋盘上所有英雄（含位置/HP/法力/星级/装备/燃烧状态）、回收槽中所有英雄

## 敌方金币机制

- 击杀敌方获得金币 = 基础值 × (敌方星数 + 1)
- 基础值：战士 80 / 法师 60 / 辅助 30 / 刺客 30 / Boss 200

## 关卡机制

| 关卡 | 敌方配置 |
|------|---------|
| 第1关 | 每种类型各 1 个（0 星） |
| 第2关 | 每种类型各 2 个（0 星） |
| 第3关 | 每种类型各 3 个（0 星） |
| 第4关 | 每种类型各 2 个（2 星） |
| 第5关 | 每种类型各 1 个（3 星）+ Boss |

- 只有一方**全部阵亡**时战斗才结束
- 击败所有敌方单位通关，获得金币奖励
- 通关后场上英雄自动回收到回收槽（治疗 20 HP，清空技能条）
- 失败扣除 HP（剩余敌人数 × 20），但仍可获得部分金币
- 进入第5关前额外获得 **800 金币**补助
- HP 归零则游戏结束

## Boss（第5关）

| 属性 | 数值 |
|------|------|
| HP | 500 |
| 攻击 | 20 |
| 移速/攻速 | 60/60 |

Boss 拥有**双技能条**：

- **基础技能**（3 点法力）：攻击全体玩家英雄，各造成 10 点伤害
- **进阶技能**（5 点法力，橙色圆点）：攻击米字型四列路径（同行、同列、两条对角线）上的全体英雄，各造成 30 点伤害

mkdir build && cd build
